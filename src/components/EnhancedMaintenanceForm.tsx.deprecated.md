# ⚠️ Formulaire de Maintenance - Déprécié

Ce composant (`EnhancedMaintenanceForm`) est **déprécié** et sera remplacé par le formulaire unifié d'intervention.

## Migration vers le nouveau système

Utilisez plutôt : **`TechnicianInterventionForm`** (`/intervention`)

Le nouveau système unifié combine :
- ✅ Formulaire d'intervention numérique
- ✅ Génération automatique de devis
- ✅ Signatures (technicien + client)
- ✅ Géolocalisation GPS
- ✅ Validation admin
- ✅ Historique des actions

## Modèle unifié

Toutes les fonctionnalités sont maintenant dans le modèle **`Intervention`** qui remplace à la fois `Intervention` et `MaintenanceReport`.
